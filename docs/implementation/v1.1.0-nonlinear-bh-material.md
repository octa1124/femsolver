# `v1.1.0` Nonlinear B-H Material Implementation

## Scope

`v1.1.0` adds the first executable nonlinear magnetic material slice. It is intentionally local-material focused: the code can evaluate nonlinear B-H response and tangent data, but it is not yet wired into the global nonlinear magnetostatic Newton solve.

## What It Can Do

- Represent a monotone piecewise-linear B-H curve starting at the origin.
- Evaluate `H(|B|)`, secant reluctivity, differential reluctivity, and magnetic energy density.
- Represent a local-frame orthotropic material by assigning independent B-H curves to the `x`, `y`, and `z` material axes.
- Evaluate component-wise `H(B)`, diagonal consistent tangent, and energy density.
- Validate the tangent against centered finite differences.

## What It Cannot Do Yet

- It does not assemble a nonlinear global residual or Jacobian.
- It does not rotate local material axes from mesh/case metadata.
- It does not perform Picard/Newton nonlinear solve updates on machine cases.
- It does not model hysteresis, loss, temperature dependence, or laminated homogenization beyond independent orthotropic axes.

## Algorithm

`MonotoneBhCurve` stores sorted samples `(B_i, H_i)`. Queries use the active segment:

- `H(B)` is linear interpolation or last-segment extrapolation.
- `dH/dB` is the segment slope.
- `nu_sec = H(B) / B`, with the initial differential slope used near zero.
- energy density is the trapezoidal integral of the same piecewise-linear curve.

`OrthotropicBhMaterial` applies the scalar curve independently on each signed component:

```text
H_i(B_i) = sign(B_i) H_axis_i(|B_i|)
K_ii = dH_axis_i / dB (|B_i|)
K_ij = 0 for i != j
```

This keeps the residual and tangent derivable from one energy model for the first orthotropic MVP.

## Code Path

- Public API: `include/femsolver/material/bh_curve.hpp`
- Implementation: `src/material/bh_curve.cpp`

## Verification

The unit tests check:

- monotone validation and rejection of invalid curves
- interpolation, secant, differential tangent, and energy values
- orthotropic response with different axis curves
- consistent tangent against finite differences

## Handoff

The next nonlinear step is to connect this local material law to tetra `Nedelec` magnetostatic cell residuals, then reuse the existing nonlinear policy hooks for Picard warm start and damped Newton iterations.
