# `v1.3.0` Hexahedral And High-Order H1 Foundation

## Scope

`v1.3.0` adds the first hexahedral and high-order H1 building blocks. The purpose is to prevent the kernel from becoming tetra-only or first-order-only while keeping the implementation small and testable.

## What It Can Do

- Represent the tensor-product reference hexahedron `[-1, 1]^3`.
- Evaluate hexahedral Gauss-Legendre quadrature rules of order 2 and 3.
- Evaluate `H1` hexahedral `Q1` basis values and gradients.
- Evaluate `H1` hexahedral `Q2` basis values and gradients.
- Verify partition of unity, nodal interpolation, gradient consistency, and constant integration.

## What It Cannot Do Yet

- It does not yet include a physical hexahedral mesh container.
- It does not yet assemble hexahedral stiffness or mass matrices.
- It does not yet include high-order tetra `P2`.
- It does not yet include high-order Nedelec or Raviart-Thomas spaces.

## Algorithm

`Q1` uses trilinear vertex functions:

```text
N_a(xi, eta, zeta) = 1/8 (1 + s_a xi)(1 + t_a eta)(1 + u_a zeta)
```

`Q2` uses tensor products of the 1D quadratic Lagrange functions on nodes `[-1, 0, 1]`:

```text
l0(x) = 0.5 x (x - 1)
l1(x) = 1 - x^2
l2(x) = 0.5 x (x + 1)
```

Gradients are produced by differentiating only the active tensor component.

## Code Path

- Reference element: `include/femsolver/kernel/reference/reference_hexahedron.hpp`
- Quadrature: `include/femsolver/kernel/quadrature/hexahedron_quadrature.hpp`
- Basis: `include/femsolver/kernel/basis/lagrange_hex_hexahedron.hpp`
- Tests: `tests/unit/kernel_hex_high_order_test.cpp`

## Verification

The unit tests check:

- reference hexahedron containment and volume
- `Q1` partition of unity and vertex interpolation
- `Q2` partition of unity, center-node interpolation, and zero gradient sum
- order-2 and order-3 quadrature weight sums equal reference volume `8`

## Handoff

The next hex step is a minimal physical hexahedral mesh and geometry mapping, followed by `Q1/Q2` scalar assembly. Hex Nedelec remains a later electromagnetic extension, not part of this foundation slice.
