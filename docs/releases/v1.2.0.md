# v1.2.0 Release Summary

## Completed

- Lowest-order tetrahedral Raviart-Thomas reference basis.
- Face-flux interpolation matrix check.
- Constant-divergence and divergence-integral benchmark.
- `motor_check --hdiv-benchmark` and CTest coverage.

## Not Completed

- Physical H(div) Piola mapping.
- Signed global face DoF map.
- Flux-conforming post-processing or mixed assembly.

## Known Risks

- The RT implementation is reference-element validated only; it is not yet a full mesh-space implementation.

## Compatibility Changes

- `motor_check` accepts `--hdiv-benchmark` and default checks include H(div).

## Next Version Carryover

- Add hexahedral/high-order H1 foundation in `v1.3.0`.
