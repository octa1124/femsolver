# v1.3.0 Release Summary

## Completed

- Reference hexahedron on `[-1, 1]^3`.
- Hexahedral Gauss-Legendre quadrature rules of order 2 and 3.
- Hexahedral H1 `Q1` basis and gradients.
- Hexahedral H1 `Q2` basis and gradients.
- Tests for partition of unity, interpolation, gradient consistency, and quadrature volume.

## Not Completed

- Hexahedral mesh container and physical geometry mapping.
- Hexahedral assembly.
- Tetrahedral `P2`.
- High-order Nedelec or Raviart-Thomas elements.

## Known Risks

- The current hex path validates reference mathematics but not physical mesh quality or global assembly.

## Compatibility Changes

- New public kernel APIs for hex reference, hex quadrature, and hex H1 basis functions.

## Next Version Carryover

- Establish coupled-field operator foundation in `v2.0.0`.
