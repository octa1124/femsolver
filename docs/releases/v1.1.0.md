# v1.1.0 Release Summary

## Completed

- Monotone piecewise-linear nonlinear B-H curve representation.
- Field intensity, secant reluctivity, differential reluctivity, and magnetic energy density evaluation.
- Orthotropic local-frame material response from independent axis curves.
- Consistent tangent validation against centered finite differences.

## Not Completed

- Global nonlinear magnetostatic residual/Jacobian assembly.
- Picard/Newton machine solve using the nonlinear material law.
- Local material-axis rotation from case metadata.

## Known Risks

- The material model is energy-consistent for the local orthotropic MVP, but machine-level convergence behavior is still untested.

## Compatibility Changes

- New public material API under `include/femsolver/material/bh_curve.hpp`.

## Next Version Carryover

- Add RT/H(div) flux-conforming foundation in `v1.2.0`.
